# 花生糖產地標示調查 — Conjoint webform

食品加工供應鏈產地標示的 CBC（choice-based conjoint）線上問卷。

- **兩宣稱情境**：標榜「產地：台灣」／標榜「日本進口」（between-subjects，進場隨機順序）
- **四產地環節**：原料來源、製造地、分裝地、品牌來源國（台/中/日/印度/未標示）
- **其他屬性**：價格、可查證性、認證
- **forced binary**（無「都不買」）：以「你覺得消費者會選哪一款」預測框架呈現
- **視覺**：供應鏈流程圖（台灣紅／進口藍／未標示橘），上下比較
- **品質檢核**：trap 題、鏡像重複、topRatio、rtMs 卡秒、內建

設計來源：`../analysis_output/design_memo.md`（v04c）、`../design/`（限制性正交設計）。
